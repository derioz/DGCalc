## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dark Premium UI** | ElevenLabs-inspired design with glassmorphism, floating orbs, and ambient particle effects |
| ⚡ **Instant Calculations** | Real-time running totals update as you tap — no lag, no loading |
| 🧾 **Receipt Modal** | Full order receipt with order numbers, per-item breakdowns, and grand total |
| 🔥 **Combo Deals** | Pre-built combo meals with purple-tinted cards and deal badges |
| 📱 **Fully Responsive** | Pixel-perfect on phones, tablets, and desktops — landscape too |
| 🔊 **Sound Effects** | Satisfying Web Audio API bleeps and bloops on every interaction |
| 🎉 **Confetti Celebrations** | Hit 10 items and watch the confetti rain |
| ⌨️ **Keyboard Shortcuts** | Press `Esc` to clear the order or close modals |
| 💫 **Micro-Animations** | Ripple effects, floating +1 indicators, number pops, and card pulses |
| 🏷️ **Breakdown Chips** | Live order summary chips in the sticky footer |
